Vuela!

Vuela! es un juego serio de género arcade y puzles en el que el jugador se pondrá en la piel de un viajero o de un trabajador de un aeropuerto.

❮
❯

Consta de tres minijuegos, donde el jugador se encargará de hacer una maleta con todo lo necesario y lo permitido para un viaje, otro desde el punto de vista de un trabajador que tiene que decidir si el viajero puede subir al avión y otro donde se plantea un test para ver si el jugador ha comprendido todo lo que el juego quiere enseñarle.

Este proyecto está creado usando Unity y C# como lenguaje. Además se hace uso de uAdventure, una plataforma dedicada a crear juegos clásicos de aventura con propósito educativo.
